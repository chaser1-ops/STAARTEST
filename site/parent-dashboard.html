Parent Dashboard

This page gives you a quick snapshot of your child's study progress across all grade levels. All data is stored locally in this browser — nothing is sent to any server.

How Progress Tracking Works

What gets tracked

  • Questions answered and accuracy percentage
  • Missed questions saved for replay
  • Study days logged
  • Confidence check-ins
  • Badges earned
  • Best correct-answer streak

How to use it

  • Visit each grade's HQ page to see detailed progress
  • Check the "Needs Work" section for skill gaps
  • Use the confidence log to track how your child feels
  • Print the Victory Certificate to celebrate milestones
  • Data stays on this device — use the same device each session

Tips for parents

Short sessions win. Confidence matters more than perfection. Celebrate the effort.

  • Do 10–15 minute sessions, not marathon sessions.
  • Start with one cheat sheet, then 2–3 practice questions.
  • Let your child explain the answer out loud — that locks it in.
  • Circle wins. Don't make every mistake feel huge.
  • Repeat the same core moves for a few days before adding new material.
