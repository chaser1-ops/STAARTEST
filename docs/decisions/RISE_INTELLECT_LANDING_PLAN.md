# Rise Intellect Landing Page Plan — 2026-04-04

## Purpose
Plan a future landing page for riseintellect.com — the umbrella platform that houses RiseIQ and STAAR Power Portal.

## NOT Building Now
This is **documentation only**. No website is being built for riseintellect.com in this phase. The focus remains on the core products (STAAR Power Portal web + RiseIQ app).

## Recommended Structure

### Page 1: Landing (riseintellect.com)
- Hero: "Rise Intellect — Tools that make learning feel powerful"
- Value proposition: K-6 learning platform with web and mobile
- Product cards:
  - **STAAR Power Portal** (web) — link to staartest.netlify.app
  - **RiseIQ** (iOS app) — link to App Store or riseiq.app
- Trust section: "Brought to you by RISE Studio Labs"
- School/community section: "Built with Harmony Science Academy in mind, designed for every student"
- Footer: risestudiolabs.com credit

### Technical Approach
- Static HTML/CSS/JS (same as STAAR Power Portal)
- Host on Netlify (separate site, linked to a future repo)
- Or host on GoDaddy directly as a simple landing page
- No backend needed — just a brand/routing page

### Audience
- Parents discovering the platform
- Schools evaluating the product
- Partners or sponsors exploring collaboration
- App Store / web visitors wanting to understand the ecosystem

### Relationship Map
```
Rise Intellect (riseintellect.com)
├── STAAR Power Portal (staartest.netlify.app) — K-6 web study platform
├── RiseIQ (riseiq.app) — iOS study app
└── Future products under the Rise Intellect umbrella
```

### Brand Voice
- Confidence-first, mission-driven
- "Learning should feel like a power-up"
- Professional but warm — not corporate, not childish
- Community-oriented — "free for families" messaging when sponsors exist

### When to Build
- After RiseIQ is on TestFlight or App Store
- When there's a need to route traffic between products
- When a sponsor/partner needs a professional umbrella landing page
