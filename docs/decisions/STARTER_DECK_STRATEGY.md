# Starter Deck Strategy — Phase 13

## Philosophy
Ship a smart small set now. Don't wait for exhaustive content coverage. The system is data-driven — new decks are trivial to add later.

### Total: ~15 decks, ~100 flashcards

## Card Content Priority
1. **Strategy reminders** (how to approach a question type)
2. **Key concept definitions** (what IS main idea, what IS a fraction)
3. **Memory tricks** (P.A.W., R.U.N., hints)
4. **Quick facts** (formulas, patterns, rules)

## What's NOT In Starter Decks
- Individual vocabulary words (future expansion)
- Per-question flashcards (requires tracking integration)
- Custom/user-created cards (requires account system)
- Spaced repetition scheduling (future complexity)

## Expansion Path
1. **Phase 14+**: Add vocabulary decks (10-20 words per grade)
2. **Phase 15+**: Add weak-spot auto-generation (from missed questions)
3. **Future**: Custom deck creation for signed-in users
4. **Future**: Spaced repetition for tracked cards
